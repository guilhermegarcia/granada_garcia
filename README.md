# Seminar on corpus phonology

A big part of our seminar will be based on R and some text manipulation. The best way to make the most of it is to follow along copying and testing the code used in class. If you wish to do that, follow the steps below:

1. Go to [posit.cloud](https://posit.cloud) and create an account. This is the tool we'll use to work with R
2. Then create a New Project from Git Repository
3. Paste the repo address: `https://github.com/guilhermegarcia/granada_garcia.git`
4. Now, click on `start_here.R` and run all the lines to install the necessary packages

You're now ready!

You will find the slides for the seminar here too (`seminar_garcia.pdf`).

## FAQ

- How do I run a line? _Place your cursor on the line and then press Ctrl+Enter (or Cmd+Enter on a Mac)_
- How do I know if the installation was successful? _`Tidyverse` can take a couple of minutes do install, and at the end of the installation, you will get a message and warnings, but you should not get any errors for any of these packages. The other packages should install very quickly._
- I couldn't install `Fonology`: what happened? _Try `remotes::install_github("guihermegarcia/fonology")`
